MyCitySelector plugin for WordPress. Detect user location, substitute any data depending on user location and much more...

== Description ==
The plugin is designed to determine the user's location, whether it is a country, region, state or city. Depending on the selected geolocation, the plugin allows you to replace any data on the site.
Includes a site widget for choosing a geolocation.
[Documentation (in development)](https://www.mycityselector.com/docs)

== Installation ==
System requirements:
PHP 7.4 or later with MySQL PDO extension installed

1. Upload `mcs` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin using `MyCitySelector` menu in sidebar.
4. Add MyCitySelector widget to your website.
